# Typora 1.9.5 Windows 安装激活指南

本项目提供 Typora 1.9.5 版本的安装包及激活工具，帮助用户在 Windows 系统上免费使用 Typora 的全部功能。

---

## 📦 包含文件

| 文件名                       | 说明                        |
| ---------------------------- | --------------------------- |
| `typora-setup-x64-1.9.5.exe` | Typora 1.9.5 安装包（64位） |
| `license-gen.exe`            | 序列号生成工具              |
| `node_inject.exe`            | 激活注入工具                |
| `README.md`                  | 本说明文档                  |

---

## 🚀 安装步骤

### 1. 安装 Typora

1. 双击运行 `typora-setup-x64-1.9.5.exe`
2. 选择安装路径（建议记住安装位置，后续激活需要用到）
3. 勾选 **"Create a desktop shortcut"**（创建桌面快捷方式）
4. 点击 **Install** 进行安装
5. **重要**：去除勾选 **"Launch Typora"**，点击 **Finish**
   - ⚠️ 激活时软件不能运行，必须确保 Typora 处于关闭状态

---

## 🔓 激活步骤

### 2. 准备激活工具

将 `license-gen.exe` 和 `node_inject.exe` 两个文件复制到 Typora 的安装目录。

默认安装路径通常为：

```
C:\Program Files\Typora
```

### 3. 执行激活命令

1. 以**管理员身份**打开命令提示符（CMD）

2. 进入 Typora 安装目录：

   ```cmd
   cd "C:\Program Files\Typora"
   ```

   > 请根据实际安装路径修改上述命令

3. 依次执行以下命令：

   ```cmd
   .\node_inject.exe
   ```

   等待执行完成后，再执行：

   ```cmd
   .\license-gen.exe
   ```

4. 复制生成的序列号：

   - 终端会显示类似 `License for you: WTWWXT-EZE598-NREHUJ-L9YSSZ` 的信息
   - 复制 `License for you:` 后面的序列号

### 4. 输入序列号激活

1. 打开 Typora 软件
2. 点击菜单栏 **帮助** → **输入序列号**
3. 随意输入一个邮箱地址
4. 粘贴刚才复制的序列号
5. 点击激活，完成！

---

## ✅ 验证激活

激活成功后：

- 点击 **帮助** → **我的许可证**，可以查看激活序列号
- 软件不会弹出激活提示窗口
- 可以正常使用所有付费功能

---

## ⚠️ 注意事项

1. **激活前必须关闭 Typora**：确保软件完全关闭后再进行激活操作

2. **激活失败处理**：
   - 打开 Typora → **文件** → **偏好设置** → **通用**
   - 勾选 **"Typora 服务使用国内服务器"**
   - 重启软件后再次尝试激活

3. **版本兼容性**：
   - 本方法仅适用于 **Typora 1.9.5** 版本
   - 经测试 **1.10.8 版本无法激活**
   - 激活成功后**不支持直接升级**，如需升级需卸载重装并重新激活

4. **备份自定义主题**：
   - 卸载前请备份自定义主题文件夹
   - 主题文件夹位置：`C:\Users\你的用户名\AppData\Roaming\Typora\themes`
   - 重装后复制回原位置即可恢复主题

5. **安全提示**：
   - 本激活方法仅供学习交流使用
   - 请在下载后24小时内删除
   - 建议支持正版软件
